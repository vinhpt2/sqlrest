ApplicationJs   _a technology to build any common application


 🌰 ApplicationJs is a No-code platform that

  • Focus on “what” Business process not “how” to code application
  • Build any common web application with a few lines of code
  • Will be the main way of Application development in software future
  • Available on cloud so any one can easy to use

 🌰 ApplicationJs is composed from

  • appForm is a configurable tool to build any user interface form
  • appWorkflow defined customer business process & help excute workflow task
  • appReport allow customer to self design their own report look and feel
  • appMobile translate appjs to native Android & iOS application with no effortness
  • appMap bring beautiful map to your application

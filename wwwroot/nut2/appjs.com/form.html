DesignUsage

 🗔 appForm is configurable window that bring database table to user interface

  • Data can come from any Odata service (PostREST...)
  • All table’s columns info (name, alias, primary key, relationship, type, require…) can be imported and refined
  • Design window with multi-tab that show data from multi-table with exactly their relationship.
  • Search, Edit, Manipulate Add, modify, delete data with window. Co-manipulate between window and map
